# Node-red node for ServerChan3( server酱3极简推送服务 )

## 使用说明

1. 访问 <https://sc3.ft07.com> 注册获得 SendKey，下载APP,大部分手机无需驻留后台可收到消息

1. 配置 `msg.sendkey` ， `msg.title` ， `msg.desp` 即可推送消息到手机
